# Demo Fallbacks

## If animation appears laggy
1. Use fewer slider moves and rely on preset buttons.
2. Disable heavy browser tabs and stay in one browser window.

## If LLM enhancement fails
1. Keep deterministic summary as primary output.
2. State that narrative was generated from locked deterministic metrics.

## If presenter deviates
1. Return to Baseline preset.
2. Replay safe sequence: Fuel Spike -> Collections Slip -> Generate Summary.
